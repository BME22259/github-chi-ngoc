## Description
This code demonstrates how to modify the flow rate of a simple syringe pump using state machine concept.
## Hardware
- ESP32-WROOM-32
- LCD 16x2
- 3 buttons with pull-up configuration
- 1.8 step angle motor
- TMC2209 motor driver
